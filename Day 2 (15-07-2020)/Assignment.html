Question 2:

Explain methods of string and array.

1.string methods:-

bold() method:The bold() method is used to display a string in bold. This method returns the string embedded in the b tag.

fontcolor() method:-The fontcolor() method is used to display a string in a specified color. This method returns the string embedded in the font tag

substr() method:-The substring() method extracts the characters from a string, between two specified indices, and returns the new sub string.

includes() method:- includes() method determines whether a string contains the characters of a specified string.Note: The includes() method is case sensitive.

repeat() method:-The repeat() method returns a string that has been repeated a desired number of times.

replace() method:-The replace() method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.

strike() method:-The strike() method is used to display a string that is struck out. This method returns the string embedded in the strike tag.

small() method:-The small() method is used to display a string in a small font. This method returns the string embedded in the small tag

italics() method:-italics() is a string method that is used to create the HTML i tag element. Because the italics() method is a method of the String object, it must be invoked through a particular instance of the String class.

2.Array methods:-

toString() method:-The toString() method converts an array to a string of (comma separated) array values.

join() method:-The join() method also joins all array elements into a string.

delete() method:-Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator "delete"

splice() method:-The splice() method can be used to add new items to an array

sort() method:-The sort() method sorts the items of an array.

map() method:-The map() method creates a new array with the results of calling a function for every array element.
